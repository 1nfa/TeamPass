Welcome to GitHub repository for Teampass.

TeamPass is a tool dedicated for managing passwords in a collaborative way such as companies. It is especially designed to provide passwords access security for allowed people. This makes Teampass really usefull in a Buisiness/Enterprise environment and will provide to IT a powerfull and easy tool for customizing passwords access depending on the user's role.

Environment

Collaborative Passwords Manager has been written in PHP/Ajax and needs an Apache/MySQL/PHP server. You must have PHP 5.3 or higher!
